The parameter "time_zone" shows in which time zone you are entering the time. Examples: 1, 0, -1 ...
The parameter "subject_advance_time" is responsible for how far in advance to send a message about the upcoming lesson (subject). Enter in minutes.
The parameter "deleting_subject_time" is responsible for when remove the message about beginning of the lesson after the start of the lesson. Enter in minutes.
